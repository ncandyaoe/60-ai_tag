竖版标签模板要求（以荷兰0草老 50×120mm 为例）
========================================

【模板提取信息】

  ⓪ 标签尺寸 — 从 .ai 页面尺寸自动提取（如 50×120mm、59×60mm）
  ⚠ 硬性要求：标签四周预留 2mm 出血位，所有内容必须在出血线内排版

【模板提取区域】从上到下共 6 个区域，从 .ai 颜色标注矩形提取坐标：

  ① title（标题区）      红色矩形  (2.1, 2.0)→(48.0, 13.0)  L型
  ② logo（品牌标区）     黄绿矩形  (40.1, 2.1)→(47.8, 7.4)  矩形
  ③ content（正文区）    蓝色矩形  (2.1, 13.5)→(48.0, 82.1) L型
  ④ net_volume（净含量区）绿色矩形  (40.0, 77.3)→(48.0, 82.1) 矩形
  ⑤ nutrition（营养表区） 紫色矩形  (2.0, 82.6)→(48.0, 110.1) 矩形
  ⑥ eco_icons（环保标区） 青色矩形  (2.0, 110.7)→(48.0, 118.0) 矩形

【L型区域 6 点坐标】

  title L型（避让右上角 logo）：
  P1=(2.1, 2.0)──────────P2=(40.1, 2.0)
  │                       │
  │   title 全宽排版区      P3=(40.1, 7.4)──P4=(48.0, 7.4)
  │                                         │
  │            title 全宽排版区               │
  P6=(2.1, 13.0)──────────────────P5=(48.0, 13.0)

  → P1~P2~P3 上方窄行宽 = P2.x - P1.x = 38.0mm
  → P6~P5   下方全行宽 = P5.x - P6.x = 45.9mm
  → P3.y    切换行 Y   = 7.4mm（logo 底边以下恢复全宽）

  content L型（避让右下角 net_volume）：
  P1=(2.1, 13.5)────────────────P2=(48.0, 13.5)
  │                                  │
  │        content 全宽排版区          P3=(48.0, 77.3)
  │                              P4=(40.0, 77.3)
  │     content 窄行排版区          │
  P6=(2.1, 82.1)────P5=(40.0, 82.1)

  → P1~P2   上方全行宽 = P2.x - P1.x = 45.9mm
  → P5~P6   下方窄行宽 = P5.x - P6.x = 37.9mm
  → P4.y    切换行 Y   = 77.3mm（net_volume 顶边以下变窄）

【各区域排版要求】

① title（标题区）
   - 字号比 content 正文大 10%，并且加粗
   - 需要先根据 content 计算出 content 的字号，然后计算出 title 的字号
   - L型排版：前几行避让 logo（宽 38.0mm），logo 下方恢复全宽（45.9mm）
   - 当位置不够时，不可突破最小字号的高度，可以执行横向压缩；

② logo（品牌标区）
   - 保持原始图像，不做文字渲染

③ content（正文区）
   - 字母 a/o/e 字高最低 1.2mm（对应最小字号 ≥ 6.1pt）
   - L型排版：上方全宽（45.9mm），底部几行避让 net_volume（窄 37.9mm）
   - 当位置不够时，不可突破最小字号的高度，可以执行横向压缩；
④ net_volume（净含量区）
   - 字高按净含量数值分级（德国强制，其他欧洲国家推荐执行）：
     | 净含量范围                    | 最低字高 | 最小字号(pt) |
     |------------------------------|----------|-------------|
     | >1000 g/mL                   | ≥6mm     | ≥23.8pt     |
     | 200 g/mL < 净含量 ≤1000 g/mL | ≥4mm     | ≥15.9pt     |
     | 50 g/mL < 净含量 ≤200 g/mL   | ≥3mm     | ≥11.9pt     |
     | ≤50 g/mL                     | ≥2mm     | ≥7.9pt      |
   - 加粗显示
   - 字号换算基于数字 cap-height 比例 0.715（Alibaba PuHuiTi 实测）
   - 当位置不够时，不可突破最小字号的高度，可以执行横向压缩；

⑤ nutrition（营养表区）
   - 字母 a/o/e 字高最低 1.2mm（对应最小字号 ≥ 6.1pt），当位置不够时，不可突破最小字号的高度，可以执行横向压缩；
   - 表格结构：粗外框 + 细行线 + 列分隔线
   - 标题行加粗，数值右对齐
   - 子项（of which / -Saturates 等）缩进

⑥ eco_icons（环保标区）
   - 图标等间距水平排列
   - 保持原始图像比例

【字高换算公式】
  最小字号(pt) = 要求字高(mm) / 比例 / 0.3528
  - 小写 a/o/e 比例 = 0.56（x-height，用于 content / nutrition）
  - 数字/大写比例 = 0.715（cap-height，用于 net_volume）
  - 以上比例基于 Alibaba PuHuiTi 字体实测，换字体需重新测量
